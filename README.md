# 20201021

バグだと思っていたのが仕様だった。
goppa多項式はバイナリ符号の場合は既約多項式でないと完全には動作しないのだという制限があるらしい。
ということで多項式の既約性判定アルゴリズムを実装しなければならない。
実を言うと以前にben-orのアルゴリズムを実装しようとしたことがあったのだが、次数が二重指数関数的に増えるので実用的でなかった。
